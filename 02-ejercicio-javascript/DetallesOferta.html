Desarrollador React

React Company Barcelona, España

Descripción del puesto

En React Company estamos buscando un Desarrollador React Junior con ganas de aprender y crecer dentro de un equipo altamente colaborativo. No se requiere experiencia previa profesional, pero sí una sólida base en JavaScript, interés por el desarrollo frontend moderno y una actitud proactiva para resolver problemas y mejorar continuamente. El candidato formará parte de un entorno ágil, donde podrá participar en el ciclo completo de desarrollo de aplicaciones web y adquirir experiencia real trabajando con tecnologías actuales.

Responsabilidades

  • Colaborar en el desarrollo e implementación de nuevas funcionalidades utilizando React.
  • Trabajar junto a desarrolladores senior para comprender buenas prácticas y patrones de diseño.
  • Participar en revisiones de código y aprender a escribir código limpio y mantenible.
  • Contribuir en la mejora continua de la experiencia de usuario y el rendimiento de las aplicaciones.

Requisitos

  • Formación en Ingeniería Informática o similar (o conocimientos equivalentes demostrables).
  • Conocimientos básicos de JavaScript, HTML, CSS y React.
  • Capacidad para trabajar en equipo, aprender rápido y adaptarse a un entorno ágil.
  • Interés por la mejora continua, la tecnología y el desarrollo frontend.

Acerca de la empresa

React Company es una empresa joven especializada en el desarrollo de aplicaciones web modernas basadas en tecnologías JavaScript. Apostamos por un entorno de trabajo dinámico, flexible y orientado al aprendizaje continuo, donde cada desarrollador tiene la oportunidad de crecer tanto personal como profesionalmente. Nuestra cultura se basa en la colaboración, la innovación y el uso de herramientas actualizadas para ofrecer soluciones de alto rendimiento a nuestros clientes.
